# Lexis Support Hub

![Lexis Logo](assets/lexis.png)

A simple, all-in-one desktop utility designed to provide quick support, troubleshooting steps, and system information for users of Lexis. Built with Python and CustomTkinter.

---

## Features

- **Interactive Support**: A command-based interface to get instant solutions for common problems like injection errors, ban issues, secure boot, and more.
- **PC Spec Viewer**: Gathers and displays detailed hardware and software specifications, including OS, CPU, GPU, RAM, and storage, with a one-click copy-to-clipboard feature.
- **Live Site Check**: Instantly check if the `lexis.re` website is online.
- **Auto-Updating**: The application automatically checks for new versions on startup and allows for a seamless, one-click update process without leaving the app.
- **Customizable UI**: Switch between Light and Dark modes and change the accent color to personalize your experience.

## Getting Started

No installation needed!

1.  Go to the **Releases** page.
2.  Download the latest `lexis-support.exe` file.
3.  Run the executable.

## For Developers

If you want to run or build the application from the source code.

### Prerequisites

- Python 3.8+

### Setup

1.  Clone the repository:
    ```sh
    git clone https://github.com/infinite-leaks/lexis-support-hub.git
    cd lexis-support-hub
    ```

2.  Install the required dependencies:
    ```sh
    pip install customtkinter Pillow requests psutil pywin32-ctypes wmi
    ```

3.  Run the application:
    ```sh
    python lexis.py
    ```
